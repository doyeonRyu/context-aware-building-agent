## 1. IBE 요소 통합 정량화 Framework
- concept
	- 여러 IBE 정량화, 각각의 영향, 합쳤을 때 영향을 정량화 해서 
	- 실측 모델을 통해 증명
- Limitation
	- 큰 novelty 없음


## 2. 실측 기반 IBE 시뮬레이션 검증 Framework
- Concept
	- 기존 IBE없이 Base model 설계
	- base model에 IBE를 넣었을 때 오차가 줄어드는 것을 증명
- Caution
	- 단순 base model이어도 어느 정도 합리적인 오차가 나와야 함
- Limitation
	- 기존 base model이 정확도가 증명되지 않으면 IBE의 영향이라고 말할 수 없음

## 3. BIM 기반 IBE 자동 추출 Framework
- Concept
	- 기존 IBE 연구는 전부 주변 건물 정보를 수동으로 입력
	- BIM에서 주변 건물 외형을 자동으로 추출해 IBE 파라미터로 변환하는 파이프라인
- Caution
	- 학술적인 새로운 novelty는 없음
	- 기술적 contribution만 존재

## 현재 목표 
> 2, 3번 믹스 

- Goal
	- 새로운 IBE 추가 시 IBE 파라미터 바로 변환해 시뮬레이션 변수로 반영 자동화
- Result
	- 이를 실측 모델을 통해 IBE 없는 상황에서 Framework를 통한 자동 IBE 반영 시 오차가 실제로 줄어드는지 증명
- Limitation
	- 학술적 novelty x

```
예상 습득 가능 데이터
- 에너지 소비량 
- 온도 / 점유 상태 / Co2 등
```

> "15분 단위 실측 데이터로 캘리브레이션된 실제 건물 모델에서, IBE 구성 요소별 영향을 에너지 총량·계절·시간대·피크 부하의 다층적 관점으로 정량화한다"

### Novelty
1. 고해상도 캘리브레이션 기반 IBE 검증 방법론
	- 기존
		- 월별/연간 총량 비교
	- new
		- 15분 단위 시계열로 캘리브레이션
		- -> IBE 추가 전후 오차 변화를 시간 해상도로 측정
		- "다른 요인 영향 때문이 아닌가"에 대한 의문을 방어
2. IBE 요소별 시간적 패턴 분해 
	- 기존
		- IBE 있음 vs 없음 총량 비교
		- 요소별 분리 정량화 
	- new
		- shading, 장파복사, 반사를 순차 추가하면서 각 요소가 어떤 시간대/계절에 오차를 줄이는지 패턴 분석
		- 물리적 매커니즘 (Shading: 일사 시간대, 장파복사: 야간)과 실측 오차 감소 패턴이 일치하면 IBE 기여분이라는 증거가 됨
3. BIM 기반 IBE 파라미터 자동 추출의 정확도 검증
	- 기존
		- 수동 입력
	- new
		- 자동 추출 vs 수동 결정 결과를 비교해서 자동화 파이프라인의 허용 오차 범위를 제시
		- 실무 적용 가능성 증명

# Research Question
**RQ1**
- IBE 구성 요소(shading, 장파복사, 반사)가 연간·월별 에너지 총량에 미치는 영향은 개별·통합 시 어떻게 다른가
	- 기존 연구 재확인 + 실측 검증
**RQ2**
- IBE 영향이 계절별/시간대별로 어떻게 달라지는가
	- 부분적 novelty
**RQ3**
- IBE가 피크 부하의 시점, 크기, 빈도를 어떻게 바꾸는가
	- 핵심 novelty, 15분 데이터로만 가능

## Novelty 3층 구조

**Layer 1 - 데이터 기반 novelty**
- 기존: 가상 건물, 월별·연간 총량 비교 당신: 실제 건물 + 실측 15분 단위 캘리브레이션
	→ IBE 시뮬레이션이 실제 건물에서 작동하는지 처음으로 고해상도로 검증

**Layer 2 - 분석 범위 novelty**
- 기존: IBE 요소를 분리하거나, 통합하거나 둘 중 하나 당신: 개별 요소(shading, 장파복사, 반사) + 통합 시 상호작용 + 계절별·시간대별 패턴 변화를 동시에 분석
	→ IBE 영향의 다층적 분해

**Layer 3 - 핵심 novelty**
- 기존: 에너지 총량 변화만 봄 당신: 피크 부하의 시점·크기·빈도 변화까지 정량화
	→ 월별 데이터로는 불가능, 15분 데이터로만 가능한 분석

# 만약 각 zone 당 co2, 온도, 점유 상태 데이터가 있다면?

## 추가되는 Novelty

**Layer 4 - 다변수 실측 기반 IBE 분리 방법론**

> "재실, 환기, 존별 온도를 실측으로 통제한 상태에서 IBE 기여분만을 분리하는 방법론"

- 실측 모델을 통해 내부 불확실성 통제
- 특정 zone 별 IBE 영향, zone의 센서 결과와 IBE 사이의 영향력 등을 증명
- 기존
	- 건물 전체 에너지 총량 비교 
- new
	- Room 단위로 IBE 영향이 공간적으로 어떻게 다른가 정량화
	- 예:
		- 남향 room: shading 효과 크고 냉방 부하 감소
		- 북향 room: shading 효과 거의 없고 장파복사 영향이 상대적으로 큼
		- 저층 room: 인접 건물 차폐 영향 최대
		- 고층 room: IBE 영향 감소
	- 실측 온도, co2, 점유 데이터 등으로 검증하면
	- IBE의 공간적 분포를 실증
