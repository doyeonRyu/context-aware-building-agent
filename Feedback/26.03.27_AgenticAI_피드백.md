## 3. Novelty — 주변 컨텍스트 반영

### 핵심 가설

> **주변 건물(컨텍스트)이 대상 건물의 에너지 성능에 얼마나 영향을 미치는가?**

- 영향 크기를 **정량화**하는 것이 연구의 핵심 과제
- 영향을 많이 받는 건물을 선별·탐색하는 프레임워크로 확장 가능
- 추후 도시 스케일 연구로의 연결점

## 4. LLM 아키텍처 개선 방향

| 항목                | 현황 / 문제                       | 개선 방향                              |
| ----------------- | ----------------------------- | ---------------------------------- |
| **RAG**           | DB 참조에 과몰입 → 창의력 저하, 토큰 소모 과다 | 의존도 축소, 보조 수단으로만 활용                |
| **프롬프팅**          | RAG 외 방법 미탐색                  | 프롬프트 엔지니어링 기법 적극 시도                |
| **Skill (MD 지침)** | 최신 기법으로 지침 파일 활용              | MD 기반 Skill 설계 구체화                 |
| **모델 선택**         | 추론 성능 vs 토큰 비용 미결정            | Reasoning 모델 vs 경량 모델 트레이드오프 분석 필요 |
